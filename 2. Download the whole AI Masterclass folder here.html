you did it! Now it is time for you to have the whole AI Masterclass folder containing the entire implementation of the Full World Model trained for Car Racing. You will find everything in the zip folder at the bottom below.

Before we do the final race we have to install all the packages required to run the Full World Model in the CarRacingGame environment. I will explain everything in detail from start to finish in the next tutorial, but please copy paste somewhere before the following line commands which we will enter together one by one in a terminal:

  1. conda create -n masterai python=3.6

  2. conda activate masterai

  3. conda install -c conda-forge tensorflow

  4. python -m pip install cma

  5. conda install -c anaconda mpi4py

  6. conda install -c kne pybox2d

  7. pip install pillow

  8. pip install gym==0.9.4

And once everything is installed, you will be ready for the final AI vs. HI competition!
